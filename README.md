# Calculator App

A simple console-based calculator application developed with Java.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Power calculation
- Square root calculation
- Exception handling for invalid operations

## Technologies Used

- Java
- IntelliJ IDEA
- Git & GitHub

## Project Structure

```text
src/
├── Calculator.java
└── Main.java
```

## How It Works

The program runs in the console and allows the user to select mathematical operations from a menu.
It takes user input with `Scanner`, performs the selected calculation, and prints the result.

## Error Handling

This project includes basic exception handling for:

- Division by zero
- Square root of a negative number
- Invalid user operations

## What I Learned

While building this project, I practiced:

- Object-Oriented Programming (OOP)
- Method design
- Exception handling
- Console input/output
- Writing cleaner and reusable Java code

## How to Run

1. Open the project in IntelliJ IDEA
2. Run `Main.java`
3. Use the menu in the console
